## 📖 About

**WorkDex** is a Progressive Web App (PWA) time tracker disguised as a Gen 3 Pokédex. Built for trainers who need to render 8 work hours a day and always forget what time they punched in.

No accounts. No backend. Just you, Charmander, and your shift.

---

## ✨ Features

- 🕐 **Live clock** — always visible at a glance
- ▶ **Time In / Time Out** — separate punch buttons
- 🏢 **Office or WFH tagging** — set your location before punching in
- 📊 **HP-style shift progress bar** — tracks your 8H goal in true Pokémon fashion
- ⏱ **Auto lunch deduction** — 1 hour unpaid lunch is automatically subtracted from net hours
- 📋 **Shift log** — all entries saved locally by date, no account needed
- 📱 **PWA installable** — add to iPhone home screen via Safari for a native app feel
- 🎮 **Gen 3 Pokédex aesthetic** — FireRed/LeafGreen palette, GBA screen, pixel Charmander sprite

---

## 📱 Install on iPhone

1. Open **Safari** and go to your GitHub Pages URL
2. Tap the **Share** button (↑)
3. Tap **Add to Home Screen**
4. Tap **Add** — done!

> ⚠️ Must use **Safari** on iOS for PWA installation. Chrome will not prompt Add to Home Screen.

---

## 🚀 Deploy Your Own

1. Fork or clone this repo
2. Enable **GitHub Pages** under Settings → Pages → Branch: `main` → `/root`
3. Visit `https://yourusername.github.io/workdex`
